FMC Route Converter

Google MyMaps CSV to GeoFS FMC

by GeoFS Flights Channel

------------------------------

This program converts a CSV file from Google MyMaps into a FMC route that can be inserted into the GeoFS FMC.

------------------------------

INSTRUCTIONS

For this program to work, make sure you are exporting only a single layer from the MyMap, and make sure that the layer contains only waypoints / markers. Any lines that are a part of the layer will likely cause the program to crash. Visit my tutorial video for more help!
